Event Bubbling and Capturing aka Trickling :-

  • The capturing phase happens before the bubbling phase

  • Event bubbling and capturing are two ways of event propagation in DOM tree

Event capturing :-

  • Event capturing is an event phase that happens before the bubbling phase. In this process, the event goes down the DOM tree from the window to the target element

Event bubbling :-

  • Event bubbling is an event phase that happens after the capturing phase. In this process, the event mover up the DOM tree from the from target element to the window.

stopPropagation() method :-

  • The stopPropagation method is available in an event parameter object. When called in an event handler, it stops the bubbling process.

stopImmediatePropagation()

  • The stopImmediatePropagation() method is available in an event parameter object. When called in an event handler, it stops other event handlers on the element from being executed and prevents the event from bubbling up.
